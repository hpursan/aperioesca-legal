# Aperioesca Legal Documents

This repository hosts the legal documents for Aperioesca - The Anti Calorie Tracker.

## Documents

- [Privacy Policy](privacy.md)
- [Terms of Service](terms.md)

## Hosted URLs

Once GitHub Pages is enabled, these documents will be available at:
- Privacy Policy: `https://YOUR_USERNAME.github.io/aperioesca-legal/privacy.html`
- Terms of Service: `https://YOUR_USERNAME.github.io/aperioesca-legal/terms.html`

## Setup

1. Create a new repository on GitHub named `aperioesca-legal`
2. Push this code to the repository
3. Enable GitHub Pages in repository settings (Settings → Pages → Source: main branch)
